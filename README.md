# Obsidian Theme ++

Obsidian based theme "obsidian-colors-syntax" but more options like "Atom Dark".
Test with: plsql, java, xml, jsp.

## Installation

Go to Settings -> Install, search for obsidian-j-syntax in Themes and click install.
